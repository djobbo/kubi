# Corehalla

## Prerequisites
- Bun
- Docker

Install dependencies:
```bash
bun install
```

Start the developement server:
> Create an `.env` file using `.env.example` as a template
```bash
bun dev
```

## Technologies
- Frontend
    - Logic & Templating
        - Typescript
        - React
    - Styling
        - TailwindCSS
        - PandaCSS
    - State Management
        - Tanstack Query (Server State)
        - Zustand (Client State)
    - Routing
        - Tanstack Router
    - Internationalization
        - LinguiJS

- Backend
    - Backend for Frontend
        - Tanstack Start (Server Functions)

- Database
    - Postgres
    - Drizzle ORM

- Build Tools
    - Vite
    - Bun
    - Docker

- Linting
    - ESLint
    - Prettier
    - Knip (Dead Code Elimination)
